## Usage
```
Simulates the flow of water

Usage: water-simulation [OPTIONS] <IMAGE>

Arguments:
  <IMAGE>  Location of the image

Options:
  -f, --framerate <FRAMERATE>  Framerate of the output video [default: 30]
  -h, --help                   Print help
  -V, --version                Print version
```
